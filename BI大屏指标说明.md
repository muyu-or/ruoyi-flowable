# BI 大屏指标说明

> 大屏每 60 秒自动刷新，支持"本周/本月"切换。部分面板每 30 秒轮播。

---

## 一、左上区域

### 1.1 任务完成率

| 指标 | 含义 |
|------|------|
| **已完成数** | 全公司在选定时段内完成的节点任务总数 |
| **完成率** | 已完成 / 全部任务（含待办、进行中、已完成、不通过） |

### 1.2 风险雷达（轮播页 1）

| 指标 | 含义 |
|------|------|
| **预警总数** | 所有节点触发的超时预警累计次数 |
| **平均响应** | 从预警产生到节点实际完成，平均花了多少天 |
| **未处理** | 预警触发后节点还没完成的数量 |
| **雷达图** | 按节点展示风险程度，面积越大 = 该节点超时越严重 |

### 1.3 预警响应明细（轮播页 2）

按节点列出：响应天数 + 预警次数。一目了然哪个节点最容易超时。

---

## 二、顶部状态栏（卡片）

| 卡片 | 含义 |
|------|------|
| **各节点 N/M** | N = 当前活跃（待办+进行中），M = 已完成。例如 `2/48` 表示 2 个在做、48 个已完成 |
| **准时率** | 在所有已完成任务中，没有触发超时预警的比例 |
| **流转效率** | 各节点平均完成比例，越高说明流程越通畅 |

---

## 三、中间主区域（生产监控）

| 指标 | 含义 |
|------|------|
| **水球图** | 完成率的可视化展示，水面高度 = 完成百分比 |
| **已完成任务** | 选定时段内完成的任务总数 |

### 轮播指标 Set A

| 指标 | 含义 |
|------|------|
| **准时完成率** | 没有超时的已完成任务占比 |
| **日均完成数** | 已完成数 / 天数（本周÷7，本月÷30） |
| **班组达标率** | 准时率 >= 80% 的班组占比，越高说明各班组超时越少 |

### 轮播指标 Set B

| 指标 | 含义 |
|------|------|
| **流转效率** | 各节点已完成占比的平均值 |
| **平均节点耗时** | 所有节点 P50 耗时的平均值，一个节点通常要多久 |
| **最快班组** | 平均处理时间最短的班组 |

---

## 四、右上区域

### 4.1 产能概览（轮播页 1）

| 指标 | 含义 |
|------|------|
| **准时完成率** | 没有超时预警的已完成任务占比 |
| **日均完成数** | 每天平均完成几个任务 |

### 4.2 库存概览（轮播页 2）

| 指标 | 含义 |
|------|------|
| **库存总量** | 当前在库物料的总数量 |
| **库存分布** | 按物料大类（原料/产品/废料）展示各类占比 |

### 4.3 经营指标（轮播页 1）

| 指标 | 含义 |
|------|------|
| **班组达标率** | 准时率 >= 80% 的班组占比，越高说明各班组超时越少 |
| **流转效率** | 各节点已完成比例的平均值 |
| **入库数量趋势** | 按日期展示入库数量变化曲线 |

### 4.4 物料周转分析（轮播页 2）

| 指标 | 含义 |
|------|------|
| **总入库批次** | 选定时段内有多少天发生了入库 |
| **日均入库量** | 平均每天入库多少件 |
| **物料子类数** | 当前库存中有多少种不同的物料子类（如玻璃、光纤、模具、电缆等） |

---

## 五、左下区域

### 5.1 节点瓶颈 Top（轮播页 1）

| 指标 | 含义 |
|------|------|
| **P50 耗时** | 50% 的任务在这个时间内完成（中位数），代表正常速度 |
| **P90 耗时** | 90% 的任务在这个时间内完成，代表最慢情况 |

> P50 和 P90 差距大 = 该节点偶尔会卡很久，需要关注。

### 5.2 员工完成排行 Top5（轮播页 2）

按完成任务数量排名的前 5 名员工。

---

## 六、中下区域

### 出入库趋势与相关性

| 指标 | 含义 |
|------|------|
| **入库/出库曲线** | 每天的入库和出库数量 |
| **相关系数 r** | 入库和出库的联动程度。r 接近 1 = 入多出也多（供需匹配），r 接近 0 = 无关联 |

---

## 七、右下区域

### 班组效率 Top

| 指标 | 含义 |
|------|------|
| **准时率** | 该班组已完成任务中，未触发超时预警的占比。越高 = 越少超时 |
| **均值耗时** | 该班组完成一个任务的平均用时 |
